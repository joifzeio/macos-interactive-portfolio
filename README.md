# Nathan's macOS Portfolio

A high-performance, interactive macOS-style portfolio built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

## 🚀 Key Features

- **Interactive macOS Desktop**: A fully functional environment with draggable, resizable multitasking windows.
- **Desktop Shortcuts**:
  - **Saveur Ramen**: A modern Japanese restaurant web experience.
  - **Soirée Inter-Facs**: An event management and ticketing platform.
- **Integrated Browser (Safari)**: An internal iframe-based browser that opens external websites natively.
- **Functional Dock & Menu Bar**: Shortcuts to major apps like Finder, Safari, Messages, VS Code, and Terminal.
- **Mobile Responsive**: Dynamically scales for a premium experience on any device.

## 🛠️ Technology Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Analytics**: Vercel Analytics

## 📁 Project Structure

- `components/apps/`: Individual application components (Safari, Finder, etc.).
- `components/dock.tsx`: The animated macOS dock.
- `components/macos-desktop.tsx`: The main desktop orchestrator and state manager.
- `public/`: Multimedia assets and static icons.
